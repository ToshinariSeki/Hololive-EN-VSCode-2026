# Change Log

All notable changes to the hololive-en-vscode-2026 extension will be documented in this file.

## [2.0.0]
- Complete contrast and readability overhaul for all existing themes (Gawr Gura, Mori Calliope, Ninomae Ina'nis).
- Created Ceres Fauna theme.
- Created 15 new themes covering all remaining Hololive EN talents: Takanashi Kiara, Amelia Watson, IRyS, Tsukumo Sana, Shiori Novella, Koseki Bijou, Nerissa Ravencroft, FUWAMOCO, Elizabeth Rose Bloodflame, Gigi Murin, Cecilia Immergreen, and Raora Panthera.
- Balanced FUWAMOCO theme elements precisely in half between Fuwawa (pastel blue) and Mococo (pastel pink) to represent the twin duo.
- Created high-contrast Light Mode support for all 19 themes.
- Updated manifest and documentation for publisher `ToshinariSeki`.
